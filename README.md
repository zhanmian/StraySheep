# StraySheep
个人博客项目 Spring Boot + MyBatis + FreeMarker + Shiro + Elasticsearch

## 技术使用

技术 | 说明 | 官网
----|----|----
Spring Boot | 容器/Java开源框架/MVC框架 | 
Shiro | 认证和授权框架
Mybatis | ORM框架
Elasticsearch | 搜索引擎
FreeMarker | 模板引擎
Druid | 数据库连接池
